                        <div class="txt js-content ptag"><p>新年快乐。下章应该就完结了。</p> 
<p><br /></p> 
<p>.</p> 
<p>青春期的孩子，对童话这东西，多半要比成年人更嗤之以鼻些，总觉得无非是无视了逻辑和实际问题，简单粗暴地把男女主人公捏在一块儿，莫名其妙就永远幸福快乐生活下去的一种体裁。然而花京院从各方面来说大概都是青春期少年中的一个例外。他当然也是听着童话故事长大的，法皇没有觉醒之前，承太郎每天睡前都会给他念故事；上了小学之后，监护人则会根据他的喜好给他购置新的童话书，看过听过的，较之同龄人都算多。花京院喜欢童话，留下的印象也很深，而且很特别，在一串串的“永远幸福快乐”之间，他总是记得些别的东西。</p> 
<p>王尔德，安徒生，快乐王子，海的女儿，就算是写给孩子看的作品也并非全都是没头没脑的皆大欢喜终成眷属，在花京院的眼中看来，童话故事中也多的是一厢情愿和自取灭亡。</p> 
<p>因此告白之前，他当然也设想过后果，承太郎或许会惊讶，迟疑，推诿，回避，甚至不容分说，直接拒绝，这些都在他的设想之内，也在常理之中——虽说若真的发生，他也不知道该如何应对，或许来一次情感爆发，或许对承太郎以死相挟，又或许其实在他的潜意识中，根本就没有准备过“会被承太郎拒绝”这样的选项。</p> 
<p>然而潜意识归潜意识，监护人的反应出乎他意料，温柔而真实的吻接连在嘴唇落下，虽然一开始采取主动的是他自己，高中生在得到回应的时候却一下子惊呆了，他难以置信对方会这样轻易地向自己抛出橄榄枝，而这橄榄枝在他眼中简直成了救命稻草，自然急不可耐地一把伸手抓住，生怕迟了几秒，对方就会变了主意。</p> 
<p>只是这样一来，故事突然变了走向，放在童话里，互诉衷肠之后便该是结局了，接着就是永远幸福快乐，再加一个干净利落的句号。然而毕竟没有任何一个童话故事教过这过于理想化的结局应该怎样到达，花京院于是僵在原地，不再说什么做什么，他等着承太郎的行动，有些不负责任地心想：你可是大人，你该教我怎么做的。</p> 
<p>法皇的触手还松松散散地堆在承太郎身边，他悄悄地收回来，尽量装得若无其事，仿佛刚才的一连串吻当中，头一个并没有伴以自己的半强迫行为。承太郎的眼睛正盯着他，两块澄澈又锐利的绿宝石一动不动，于是他也不敢移开视线，直直地，赌气似的盯回去，不甘示弱，脸颊和耳朵却都升起一阵阵让人心跳加速的热度，控制不住——才刚刚发生过那么重要的事，他现在怎么也没法装出什么平静的样子来。</p> 
<p>承太郎看了他一会儿，接着问道，电影还看不看了？</p> 
<p>“啊？哦，那，看，看吧。”</p> 
<p>他没料到对方会问出这么个问题来，答案未过脑子，结结巴巴，但是脱口而出，承太郎于是按下遥控器又关了灯，客厅陷入了一片黑暗，对面的屏幕上先出现了制片公司的台标，雪山周围飞过一圈小小的星星，与此同时承太郎的后背靠上了沙发，他盯着屏幕，竟然真的看起电影来了。</p> 
<p>花京院心想见鬼，我刚才就不应该直接答应，这搞的哪出？正懊恼得想要踹茶几，一只手顺着沙发面悄没声息地移动过来，皮肤热得发烫，动作却毫不犹豫，承太郎勾过他的腰，把他按在自己身上，熟极而流一气呵成地歪过头来，亲了亲他的耳朵尖，叹一口气说：</p> 
<p>“真是好久没这样抱你了。”</p> 
<p>那只手隔着薄薄一层棉T恤，揉他的腰，把他锁在了怀里。花京院刚才还觉得锁骨以上的皮肤全都火烧火燎，现在热过了头，反而觉得像是凉了，不仅凉，还伴着后颈头皮指尖四肢无处不在的麻，浑身电流乱窜，他几乎要哆嗦了，然而骨头缝之间又蔓延起一片说不上来的酥，搞得他连哆嗦的力气都提不起来。高中生战战兢兢许久，最后试探性地伸出手臂，一只手从沙发和承太郎的缝隙之间摸过去，另一只爬过对方的腹肌和肋骨，他抱住承太郎，心乱如麻，气息紊乱，那心态活像东亚的杂技表演者一点一点地将头塞进鳄鱼的嘴巴。然而正如杂技总是有惊无险，承太郎当然没有躲，没有推开他，没有质问你在干什么，他们现在的关系不再是单纯的孩子和监护人了，更为复杂的关系当然也允许更为亲密的互动。</p> 
<p>他抱过去之后，为求安心，慢慢地越缠越紧，承太郎轻轻地笑了笑，花京院于是抬头将视线从屏幕那里转过来。年长的恋人没有低头看他，一片黑暗中只有一小半脸被照亮，承太郎原本轮廓就深，这样强烈的光阴对比更显得他像尊雕塑，只是比起石头里剥离出来的塑像，他更鲜活生动，有温度，也有颜色。</p> 
<p>花京院心想，承太郎竟然脸红了。</p> 
<p>而他自己和承太郎比起来，当然也好不到哪里去。</p> 
<p>&nbsp;</p> 
<p>这电影看得云里雾里，与其说是看了部电影，不如说是给花京院一个机会，心安理得肆无忌惮地在黑暗中摸索感受他的承太郎，从形体到气味，从软硬到温度，他像只好奇心过剩的小动物一样在对方怀里拱来拱去坐立难安。直到余光瞥到屏幕上一黑，转头一看，原来已经开始放电影片尾的字幕了。承太郎一只手开了灯，另一只手依旧挂在他腰上，他看看时间，说都十点多了，早点洗澡睡觉吧，花京院倚在他身上没有动，嘴里答道，嗯，今天我要睡你房间。</p> 
<p>对方犹豫几秒，大概是进行了很激烈的心理斗争，也不知道具体在这几秒中究竟考虑到了哪一步，不过无论如何，最后还是点头答应了。他让花京院先去洗澡，后者于是得以先一步湿淋淋地跳上那张久违的大床，心跳如擂鼓地等着前者擦着头发走进卧室。</p> 
<p>承太郎走到床边的时候没有穿上衣，刀刻一样的肌肉线条随着动作在身体表面很暧昧地浮动着，花京院盯着松落落的睡裤裤腰上岔出来的那两道人鱼线，他吞了口口水，心想我的条纹睡衣是不是有点不合适了。承太郎走过来看了他一眼，接着很强硬地把他从床上拽下来。</p> 
<p>“怎，怎么啦？！不是答应了让我睡这儿吗？！”</p> 
<p>“睡觉之前怎么不吹干头发。”</p> 
<p>花京院像只不听话的猫似的，被他拎到了浴室的镜子前吹头发，他很想抱怨两句：你倒是分分轻重缓急？有你这样把我从床上赶下来的吗？是不是还拿我当小孩儿？话到嘴边，吹风机猛然被打开了，热气流带着吵闹的嗡声，一下子把这句抱怨堵了回去，而等到花京院准备就绪，打算再次开口的时候，承太郎的手指又顺着湿漉漉的头发插了进来，那指腹一路滑行，弹入，隔着发丝按摩着头皮，花京院噤了声，缩着肩膀，低着头，不想看见镜子里的自己或是承太郎。</p> 
<p>他懂事早，又不太愿意在小事上依赖别人，因此什么事都很早就学会了自己做，监护人上次帮他吹头发似乎还是小学，而小学生绝不会因为这样的身体接触就满脑子不能说出来的念头，浑身发热，还稍微有点起反应。等到那嘈杂的吹风机停下来，他回过头去看着身后的承太郎，后者正在收拾电器，低头卷着弯弯曲曲的电源线，花京院踮起脚勾住他的脖子，仰头去亲他的嘴唇。</p> 
<p>这个吻很长。吹风机被扔在了洗手台上，浴室门被嘭咚一下撞开，卫生间的灯一夜都没关，承太郎箍住他的腰跌跌撞撞地抱着他回到二楼的主卧，行走期间，藕断丝连的吻一直都没有真正停下。缺氧，耳鸣，心悸，一系列反应都随着唇舌交缠出入而在体内诚实地反映出来。宇航员在太空中或许就是这么个感觉，花京院晕晕乎乎地心想，失重。</p> 
<p>承太郎把他抛在主卧的床上，吻终于暂时断开了，年长者眼神迷离，高中生也好不到哪儿去，一把伸手抓住对方的裤腰往下扯，运动睡裤的边缘被他拉了下来，露出内裤的边缘，花京院有点不耐烦，他再次伸出手握住内裤的边缘，这次没来得及得逞，承太郎掐着他的手腕，喘着气说：慢点。</p> 
<p>“我不要慢点！”</p> 
<p>高中生急得不行，他蛮不讲理，手上继续使劲，然而敌不过承太郎的臂力，正如胳膊拧不过大腿。原本就松松垮垮的睡裤一半挂在大腿上，但那条内裤边依旧坚挺地留在原处，像道难攻的防线。花京院焦躁起来，他能看到对方的这最后一道防线遮不住已经抬头的欲望，承太郎的阻止在他看来莫名其妙，毫无道理。</p> 
<p>“你也想要，我也想要，为什么要慢点？”</p> 
<p>“你不想要。”承太郎低声答道，空着的一只手将睡裤稍微向上提了一些，似乎对自己暴露了勃起的事实稍感尴尬，“至少想要的不是那个。你根本不知道那是什么。”</p> 
<p>“我知道的！”花京院坐直了身子，急于证明自己并非承太郎认识中的那种屁孩，甚至可以为此稍微牺牲一点点尊严和隐私，“我……我看过不少这种东西，我知道怎么做，我也愿意做……”</p> 
<p>因为对象是你，所以对我做什么都没关系。后半句话他终究没有说出来，然而他把这句话放进了眼神和动作，他希望承太郎能明白他的意思，能被他说服，能顺势继续到下一步，然而承太郎摇了摇头，他在花京院身边的床沿坐下来，那只钳着花京院手腕的手稍稍移动，变为十指相扣，依旧紧紧地抓着花京院那只随时准备暴起扒他裤子的手。</p> 
<p>“你不明白，你没经历过，所以不明白的。”</p> 
<p>“总会有第一次嘛！”</p> 
<p>“第一次不是现在，”承太郎循循善诱，那声音很有说服力，十年中多数时候都充当着听话好孩子角色的花京院几乎就快被他说服了，“我会伤到你的，等你稍微再长大一些，身体发育成熟了——”</p> 
<p>“这可真过分啊，”花京院打断他，“抱我，吻我，把我推在床上，把我撩起来，结果扭头就说不做。”</p> 
<p>“不做，”承太郎无视他的异议，斩钉截铁地下了结论，数秒过后又稍稍松口，改口道：“至少不做到最后。”</p> 
<p>“那最后之前的呢？”</p> 
<p>没有回答，两人对视了一会儿，似乎在互相挑衅，接着花京院就看到白金之星在视线中一闪而过。主卧暖黄色的灯光跟着啪嗒一声消失，黑暗中一只滚烫的手顺着肚子摸上来，停留在胸口正中，用不大不小的力气重新把他推到床面，花京院在一片伸手不见五指中摸索到了对方的后颈，于是用力把对方按下来贴到身上。一个吻大概是方位不准，落到了他的嘴角，又顺着下巴一路迤逦到喉头和锁骨，行进过程中扯开了睡衣纽扣，花京院发着抖，脚趾尖都紧紧地蜷起来，说不清是因为期待还是因为不安。</p> 
<p>“还是决定做了？”</p> 
<p>“不做到最后，不会让你受伤的，”那只滚烫的手沿着肋骨，沿着侧腹滑下来，滑进了睡衣的裤腰，不紧不慢，几乎是在花京院的皮肤上闲庭信步，“也有很多别的方法……”</p> 
<p>“用嘴和手？”</p> 
<p>“猜，”睡裤的温度和触感逐渐从腰臀剥离，顺着大腿，膝盖，小腿，摩擦滑落，最后在黑暗中被丢到一边，“或者也不用猜，一会儿你就知道了。”</p> 
<p>&nbsp;</p> 
<p>第二天早上睁眼的时候，他整个人懒得一动都不想动，承太郎还在睡，呼吸声近得几乎要撞进耳朵，痒痒的，不算太舒服，但又让人舍不得离开。花京院多少还有些扭过头去的力气，于是稍微转转脑袋，面对面地看向躺在身边的承太郎，浓眉，长睫毛，漂亮的鼻子和嘴，平稳，安静地呼吸沉睡，好看得像个假人。</p> 
<p>这么好看又帅气的人，他现在是我的了。花京院这么想着，心里不由自主地满是骄傲与得意。十六岁的高中生之前毕竟只谈过一次一塌糊涂的恋爱，虽说是初恋，却几乎不能算进来计数，没怎么经历过恋爱，自然也没经历过劈腿，挖角，移情别恋之类的破事，心爱的人既然已经收入怀中，自然就认定他已经成了自己的所有物，以后再也不会变了。而且不仅是自己的，还是自己打败了周围所有必然存在的成熟情敌而一举拿下的，这感觉仿佛是成了世界第一高峰的最年轻攀登者之类，不仅是最年轻，而且还是第一个征服这座高峰的人。</p> 
<p>想到这里他又皱皱眉头，不情愿地纠正刚才的想法，不，不是第一个，承太郎谈过一次恋爱的，我是第二个。</p> 
<p>这想法成功地催生出了一份莫名其妙的危机感，他扭头在那对红而饱满的唇上偷一个早安吻，承太郎像个睡美人一样被他吻醒了过来。睡美人眨眨眼睛，又紧接着为他的早安吻闭上了双眼，承太郎微笑着回了一个吻，柔软和湿热停留在嘴唇上，花京院看着他，脱口问道：</p> 
<p>“承太郎，你以前喜欢过的那个人是谁？”</p> 
<p>这问题或许是问错了，这样一个温存的清晨，明明更适合交换些甜言蜜语的悄悄话，而不是接吻过后劈头就问对方他的上一个恋爱对象。花京院能感觉到包裹着自己的手臂和身躯都因为他的提问而猛然僵硬，承太郎沉默不语，他没有看着花京院，他皱起了眉。</p> 
<p>“你要是不想说的话——”</p> 
<p>“不，这个我总该告诉你的。”承太郎依旧没有把视线转到他这边来，温暖和适当的重量从花京院身上缓缓离开，承太郎下了床，穿上拖鞋，离开了卧室，数分钟过后又回来，手里拿着那个花京院见他已经用了十年的真皮钱包。一张老旧泛黄的照片从钱包的夹层中被小心翼翼地抽了出来，递到他面前。</p> 
<p>“就是这个人。”</p> 
<p>花京院接过照片，审视着照片上模模糊糊的几个人像，他一眼认出了其中的承太郎和波鲁那雷夫，年轻，带着微笑，和现在相比变化并不很大；稍过一会儿，又后续看出了蹲在镜头前的是乔瑟夫乔斯达，他看上去比现在高大，开朗，虽然已经步入老年，却依旧精神奕奕；而照片上其余的人，他认不出来了，他也不知道那条看上去很高傲的狗是什么来历，那个黑皮肤的大概是非洲人吧，而站在承太郎身边的，穿着学生制服的——</p> 
<p>花京院皱起眉头，凑近了看着照片上的这个东方人。</p> 
<p>“他叫花京院典明。”承太郎的声音似乎是从很远很远的地方传来的，“他死的时候，应该是十七岁。”</p> 
<p>花京院盯着照片上的这个人，他站在承太郎身边，笑得很幸福，但是那笑容同时又有些矜持，眼皮上两道明显的伤疤，制服和承太郎一样是进过改造加长的款式。或许承太郎不说，他也能看出来他们当时是一对，他们挤在照片的一个角落里，挤在一起，微笑，背靠背肩并肩，哪怕知道这是一张会被留下来的，会被所有人看到的影像记录，恋人们依旧会忍不住在任何时刻都挨在一起的。</p> 
<p>花京院看着照片里的这个人很久，最后问道：他死了吗？</p> 
<p>“对，死了，最后一战的时候，我没有亲眼看到，到的时候，已经死了。”</p> 
<p>“你是因为他才领养我的对吗？”</p> 
<p>“……是。”</p> 
<p>“你说你们在一起两个月就分手了，其实不是分手，是因为他死了对吗？他不死的话，你们会一直在一起的。”</p> 
<p>“或许是这样没错。”</p> 
<p>一问一答，生涩但并无停顿，花京院盯着照片里的花京院，照片里那个十七岁的高中生冲着他微笑，这微笑模糊发黄，倾诉着他们之间所相差的时间。花京院心想，这么说，我是个替代品。</p> 
<p>他抬头问承太郎：你是拿我当了替代品了对吗？</p> 
<p>问答是具有惯性的，若之前所有的答案都是肯定，那紧接着的问题多半也会跟着得到一句“没错”，花京院想到这句可能即将到来的“没错”，心沉沉的几乎要落进腹腔，他等着答案，直到承太郎回答他：</p> 
<p>“一开始是。”</p> 
<p>“一开始？”</p> 
<p>“你还在上小学的时候，差不多是那样，但是国中以后，性格变了，就和他越差越远，不太一样了。他一直到十七岁都没一个朋友，你不一样。”</p> 
<p>“一个朋友都没有，也太惨了。”</p> 
<p>“是啊。”</p> 
<p>他们同时沉默了下来，同时看着照片上的花京院典明，一个已死之人变形变色的微笑，花京院在心中酝酿着下一问题，他有太多想问，反而又一个都问不出来了，他怕得到的答案刺激到自己，又怕自己随便问什么都伤害到承太郎，他能感觉到承太郎在微微颤抖，他虽然什么也没说，但是平静之下酝酿着某种快要爆发的情绪。</p> 
<p>“承太郎——”</p> 
<p>他想说今天先不谈这个吧，也是他不好，如果他不提起这茬，这本来应该是个黏黏糊糊的，甜蜜的假日，他们完全可以多享受几天亲热的快活日子，可现在他们却对着一张老照片支吾其词难上难下。花京院后悔了，后悔因为一时醋意毁掉这本该非常美妙的整整一天，承太郎却在他叫了他的名字之后回过头来，那对绿眼睛里闪着光：</p> 
<p>“不，你听我说。”</p> 
<p>“……你说吧。”</p> 
<p>“你们并不是同一个人，你也并不是他的替代品，”他盯着花京院的眼睛，一字一顿地说道，“我昨晚的话都是对现在的这个你说的，而他已经死了十七年了。”</p> 
<p>这四个短句他说得非常吃力，好像不仅要说服花京院，也同时要说服自己。最后一个结论掷地有声地从他嘴里说出来之后，承太郎看着那张照片不再开口，花京院莫名觉得自己知道他在想什么：承太郎大概也是今天才第一次对自己承认，他那曾经的恋人已经确确实实的死了，而且已经离开了很多年。</p> 
<p>在几分钟之前，他还酝酿着失望，震惊，愤怒和不知所措，然而承太郎坦白之后，这些心情像是被一阵风利落地刮走了一样，他坐在原处，最后一次看了看那张和自己几乎一模一样的脸。</p> 
<p>其实并不是一模一样，他对自己说，我比你年轻，我比你清晰，我在他身边，我还活着。</p> 
<p>花京院伸手抱住了承太郎的脖子。</p> 
<p>“我不在乎那些东西，”他咬着承太郎的耳朵低声说，“相差十八岁，你领养了我，我们是同性，或者你有个跟我长得一样的前男友，或者你是为了他才找到了我，我都不在乎。”</p> 
<p>“花京院——”</p> 
<p>“我等了这么久，才不会随随便便把你让给别人呢，”他把那张照片抛在身后，也抛在脑后，抱紧了怀里的监护人，启蒙老师，以及恋人：</p> 
<p>“再说，反正他现在也没法跟我抢啊。”</p>
